# Tetriste  - A Simple Tetris Game
![Tetriste Game](image-1.png)
## Description
This is a simple game project written in C++ using the SFML library. The game is called Tetriste.
![Tetriste - Hard](image.png)
## Build Project
```cmake -S src -B build```

## Run Project
```cmake --build build```

## Executable Path
```./build/bin/Tetriste.exe```
